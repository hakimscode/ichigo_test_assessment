# Online Test - Full Stack Engineer

## Test 1
Open `rewards_system` folder

```cd rewards_system```

### Usage
1. Install dependencies `npm install` or `yarn install`

2. Run `npm run dev` or `yarn dev` to run in development mode

3. Run `npm run test` or `yarn test` to run the tests

4. Run `npm run build` or `yarn build` to build the system. Output folder is in `dist`
